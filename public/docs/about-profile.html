BikeTag now lets players create a profile using simple social logins — no passwords, no hassle. Your profile reserves your unique BikeTag name and keeps track of the tags you've posted and found across games.
Ambassadors (the folks helping run local games) can also log in to access admin tools for approving new tags and keeping the game flowing smoothly.

Have questions or want help with your profile? Reach out anytime at support@biketag.org.
