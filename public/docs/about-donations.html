BikeTag is a free, open-source project built and maintained by volunteers who love bikes, code, and community. Donations help cover hosting costs, keep the app running smoothly, and support future improvements.
If you enjoy playing or want to help the project grow, you can contribute in a few ways:

  • Support us on Patreon
  • Sponsor the project on GitHub
  • Explore or contribute to the code

Every bit helps, and we’re grateful for the support!
